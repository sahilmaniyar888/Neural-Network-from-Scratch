# Fundamentals

This section focuses on first-principles PyTorch ideas.

## Notebooks

- `01_pytorch_basics_and_training_loop.ipynb`: tensors, autograd, manual training, and the transition to `nn.Module`.
- `02_sft_batch_preparation.ipynb`: supervised fine-tuning batch formatting, masking, and loss construction concepts.

## Suggested order

Start with `01_pytorch_basics_and_training_loop.ipynb` and then move to `02_sft_batch_preparation.ipynb`.
