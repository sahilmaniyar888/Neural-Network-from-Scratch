# Computer Vision Notebooks

This folder contains higher-level PyTorch notebooks that build on the fundamentals and data-loading sections.

## Notebooks

- `01_mnist_cnn_end_to_end.ipynb`: a compact CNN example that loads MNIST, trains on a small subset, and inspects predictions.
- `02_transfer_learning_with_alexnet.ipynb`: fine-tuning AlexNet on your own image dataset.

## When to use this folder

Come here after you are comfortable with:

- tensors,
- autograd,
- the training loop,
- and basic dataset/data-loader patterns.

## Practical tip

`02_transfer_learning_with_alexnet.ipynb` is more resource-heavy than the fundamentals notebooks, so Colab can be a good fit if your local machine feels slow.

If you want a faster CPU-friendly starting point, begin with `01_mnist_cnn_end_to_end.ipynb` first.
